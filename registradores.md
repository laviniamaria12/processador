# Registradores - Processador

Registradores organizados por tamanho e código binário (4 bits):

## 8 bits
- R80  = 0000
- R81  = 0001
- R82  = 0010
- R83  = 0011

## 16 bits
- R160 = 0100
- R161 = 0101
- R162 = 0110
- R163 = 0111

## 32 bits
- R320 = 1000
- R321 = 1001
- R322 = 1010
- R323 = 1011
- DESV = 1100

## 128 bits
- R1280 = 1101
- R1281 = 1110

## Flags (8 bits)
- Bit 3 (Z): Zero
- Bit 2 (N): Negativo
- Bit 1 (C): Carry
